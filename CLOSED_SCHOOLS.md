# 削除済み学校一覧

このマップから削除された学校とその理由を記録します。マップは現役校のみを表示する方針です。

## 大学 (2026年6月検証で対応)

| 削除/改名内容 | 理由 |
|---|---|
| 神戸医療未来大学(神戸キャンパス) **削除** | 実在しないキャンパス (神戸医療未来大学は姫路/大阪天王寺/神戸三宮サテライトのみ) |
| 神戸医療福祉大学(中町キャンパス) → **神戸医療未来大学(姫路キャンパス)** | 2019年に大学名変更済み、キャンパス名も姫路キャンパスに変更 |
| 兵庫県立大学(姫路書写キャンパス) → **兵庫県立大学(姫路環境人間キャンパス)** | 正式名称 (姫路書写キャンパスは存在しない) |
| 兵庫県立大学(淡路キャンパス) → **兵庫県立大学(淡路緑景観キャンパス)** | 正式名称 |
| 京都光華女子大学 → **京都光華大学** | 2026年4月に名称変更・男女共学化 |
| 宝塚大学(宝塚キャンパス) **削除** | 造形芸術学部は2016年募集停止・2021年法人本部も移転。学部生不在で実質閉鎖 |



## 1. 重複登録 (架空校名)

「市町村名 + 校名」の通称が独立校として誤登録されていたもの。実体は同一校。

| 削除した校名 | 実在する正式校 | 所在地 | 検出方法 |
|---|---|---|---|
| 大阪狭山 | 大阪府立狭山高等学校 | 大阪狭山市半田4-1510 | 同座標重複検出 |
| 河内長野 | 大阪府立長野高等学校 | 河内長野市原町2-1-1 | 同座標重複検出 |

## 3. 2027年閉校予定 (兵庫県立高校 第三次実施計画)

2025年4月に統合校が開校。被統合校はまだ在校生が残っているが、2027年3月までに順次閉校予定。教育委員会の「県立高等学校教育改革第三次実施計画」に基づく。

### 北神戸総合 (2025年開校) ← 2校統合

- 神戸北 (神戸市北区有野中町)

### 神戸学園都市 (2025年開校) ← 2校統合
- 伊川谷北 (神戸市西区学園西町6-1) ※統合校が同校地を使用
- 伊川谷 (神戸市西区前開南町)

### 西宮苦楽園 (2025年開校) ← 2校統合
- 西宮北 (西宮市苦楽園二番町16-80) ※統合校が同校地を使用

### 三木総合 (2025年開校) ← 3校統合
- 三木東 (三木市別所町小林625-2) ※統合校が同校地を使用

### 播磨福崎 (2025年開校) ← 2校統合
- 福崎 (神崎郡福崎町福田234-1) ※統合校が同校地を使用
- 夢前 (姫路市)

### 姫路海稜 (2025年開校) ← 3校統合
- 姫路南 (姫路市大津区天満191-5) ※統合校が同校地を使用
