# 関西6府県 高校・大学 所在地マップ

🌐 **公開ページ**: https://imaooo4416.github.io/koko-tizu/

## 機能

### マップ表示
- 関西6府県の高校・大学を OpenStreetMap / 地理院タイル上にプロット (実座標)
- **マーカー形状で種別を判別**: 高校 = ●○ (円) / 大学 = ◆◇ (菱形)
- 府県色で塗り分け
- 鉄道路線オーバーレイ (OpenRailwayMap)

### フィルタ
- 府県 (大阪/兵庫/京都/奈良/滋賀/和歌山)
- 市区町村 (府県別モーダル)
- 高校・大学の独立トグル
- 高校 種別 (公立/私立)
- 大学 種別 (国公立/私立)
- 鉄道路線選択 → 該当路線駅から半径1.2km以内の学校のみ
- 学校名のあいまい検索 (ひらがな・カタカナ・記号差を無視)

### 詳細表示
- 学校をクリックで詳細パネルが下からスライドアップ
  - 種別バッジ (高校 / 大学)
  - 所在地・最寄駅・路線・徒歩時間
  - 公式サイトリンク (TOPページ)
  - Google Map で表示
  - 沿線情報 (最寄駅から1.2km以内の路線一覧)
- 学校一覧は右側ドロワーに表示、上1/3=一覧 / 下2/3=詳細 の縦並びで他校に切替簡単

### 路線エフェクト
- 選択中の路線は CSS アニメーション (stroke-dashoffset + opacity pulse) で
  ドットが流れるように点滅 — GPU 合成で軽量動作

### 行政区域フォーカス
- 単一府県/市区町村選択時にポリゴンで該当エリアを切り取り、外側を半透明マスク
- 境界線をドローインアニメーション + 自動ズーム

### PDF 出力
- 表示中の学校を A4 横向きで一括 PDF 化
- **高校・大学それぞれ別ページ**、番号は各ページ1始まり
- 高校ページ: 府県×エリア(地域) ごと
- 大学ページ: 府県ごと
- 公立/私立(または国公立/私立) を凡例で区別

## 削除データの管理

廃校・架空校・改名済み校・統合により学部生不在となったキャンパス等は [CLOSED_SCHOOLS.md](CLOSED_SCHOOLS.md) に記録。
過去の検証で:
- 大阪府高校3校 (架空名2件 + 2008閉校1件)
- 兵庫県高校14校 (2025年統合校制度の被統合校・2027年閉校予定)
- 大学2キャンパス (実在しないキャンパス + 学部生不在)
- 大学4件を改名 (姫路書写→姫路環境人間、京都光華女子→京都光華 等)

## ローカル開発

ビルド不要。任意の静的ファイルサーバーで起動できます。

```bash
npx serve . -l 8765
# または
python3 -m http.server 8765
# ブラウザで http://localhost:8765 を開く
```
